# Заданиt 1: Скриншот с багами

### Приоритезация:
*   **P0 (Blocking):** Блокирует основной сценарий (поиск/выдача), требует немедленного исправления.
*   **P1 (High):** Существенно портит UX или нарушает логику фильтрации/навигации.
*   **P2 (Medium):** Заметные визуальные дефекты или имиджевые ошибки.
*   **P3 (Low):** Мелкие недочеты контента или верстки.

---


| Приоритет | Описание дефекта | Аргументация  |
| :--- | :--- | :--- |
| **P0** | **Некорректная работа фильтра по цене** | **Critical Functional Bug.** В параметрах задано 1к–30к, но в выдаче 500₽ и 40к. Фильтры поиска (основной функции) не работают. |
| **P0** | **Критическая нерелевантность гео-выдачи** | **Critical Functional Bug.** При выбранном фильтре «Москва» на первом месте объявление из Новосибирска. Цель поиска не достигнута. |
| **P0** | **Системная ошибка «Ошибка, обновите страницу»** | **Stability Bug.** Признак нестабильности фронтенда или падения микросервиса. Блокирует доверие пользователя к работоспособности сайта. |
| **P1** | **Нарушение логики гео-сортировки (Можайск)** | **Major Functional Bug.** Выбрана Москва, но предлагается сортировка «Сначала из Можайска». Нарушение регионального контекста. |
| **P1** | **Подмена функционала иконки «Избранное»** | **Major UX Bug.** Вместо добавления в избранное кнопка предлагает скрыть похожие. Ломает ключевой сценарий удержания пользователя (Retention). |
| **P1** | **Несоответствие фильтров категории (Щенок)** | **Logic Bug.** В категории «Кошки» отображается фильтр возраста «Щенок». Признак некорректного наследования атрибутов в БД/API. |
| **P1** | **Инверсия иерархии Breadcrumbs** | **Navigation Bug.** Путь `Персидская > Кошки` вместо `Кошки > Персидская`. Нарушает логику вложенности и индексацию сайта. |
| **P2** | **Рассинхрон счетчика выдачи** | **Logic Bug.** Заголовок заявляет 19 объявлений, фильтр - 8, визуально - другое количество. Вводит пользователя в заблуждение. |
| **P2** | **Грамматическая ошибка «Коръера в Авито»** | **Impairment of Reputation.** Грубая опечатка в хедере на главной странице крупного бренда подрывает имидж компании. |
| **P2** | **Грубое нарушение целостности UI-сетки** | **Visual Bug.** Наложение карточек, разная высота, карточка под углом. Сайт выглядит «разваленным» и небезопасным для сделок. |
| **P2** | **Выход виджета за границы** | **UI Bug.** Окно «Сообщения» обрезано краем экрана. Потеря части функционала управления коммуникациями. |
| **P3** | **Ошибка модерации контента (Игрушка)** | **Content Bug.** Плюшевый котенок в категории живых животных. Мелкий недочет фильтрации объявлений или проверки модераторами. |

---

### Резюме по анализу:
Страница перегружена критическими дефектами в **основной функциональности** (Поиск, Гео, Фильтры). С точки зрения качества, текущее состояние интерфейса оценивается как **Unstable**, так как нарушены базовые контракты между вводом пользователя и выдачей системы.
